# ASSINANET — Sistema de Assinatura Digital

> Solução completa para gestão de contratos e assinaturas digitais com validade jurídica, desenvolvida para provedores de internet e empresas que exigem conformidade com a **Lei 14.063/2020**.

---

## Sobre o Projeto

O **AssinaNet** é um sistema proprietário desenvolvido para modernizar e juridicizar o processo de assinatura de contratos em campo. Criado para atender às necessidades reais de provedores de internet, o sistema elimina papel, reduz retrabalho e garante segurança jurídica em cada documento assinado.

## Funcionalidades

### Documentos Suportados
- Contratos de Instalação
- Termos de Mudança de Endereço
- Ordens de Serviço Técnico

### Assinatura Digital
- Captura via touch (tablets/celulares) ou mouse
- Modo tela cheia para dispositivos móveis
- Assinatura remota por link enviado ao cliente
- Validade jurídica garantida conforme Lei 14.063/2020

### Administração
- Painel administrativo completo
- Gestão de técnicos e níveis de permissão
- Cadastro e gerenciamento de templates de documentos
- Filtros avançados e busca de contratos

### Comunicação
- Envio de documentos por e-mail com PDF anexo
- Compartilhamento via WhatsApp
- Links de assinatura com validade de 24 horas
